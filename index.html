What the number says

At the current assumed price, eque-cel improves QALYs compared with SoC, but the incremental cost is too high relative to the SAR 215K/QALY threshold. The result suggests a strong need for pricing scenarios, risk-sharing, and Saudi-specific comparator validation before using the model for payer negotiation.

Reusable POC Loop

The reusable template keeps the model logic stable, while replacing country-specific data. AI handles repetitive search and extraction work. Humans confirm high-risk assumptions.

Workflow
1. Select country Choose Saudi, UAE, Singapore, Australia, etc.
→
2. AI country scan Search payer, HTA, epi, price, guideline, comparator.
→
3. Candidate parameters AI extracts values into staging table with source snippets.
→
4. Human review Pricing, Market Access, Clinical, KOL approve key assumptions.
→
5. Run model XLSM / Python calculates ICER, BIA, sensitivity.
→
6. Report output Generate one-page summary and negotiation strategy.

Method

For each DSA-eligible parameter, the system runs the model twice: once at the lower bound and once at the upper bound. The ICER movement is calculated and ranked. Human reviewers define which parameters are eligible and whether the ranges are clinically and commercially reasonable.
